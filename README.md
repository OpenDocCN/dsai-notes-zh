### Docker

```
docker pull apachecn0/dsai-notes-zh

# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g dsai-notes-zh
dsai-notes-zh <port>
# 访问 http://localhost:{port} 查看文档
```
